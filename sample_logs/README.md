# Sample Logs

This folder contains sample log files generated by the Cowrie Honeypot.

## File

### cowrie_sample.json

This file contains sample attacker events recorded by the Cowrie SSH Honeypot.

The log entries include:

- Timestamp
- Source IP Address
- Session ID
- Event Type
- SSH Login Attempts
- Executed Commands
- Connection Information

## Purpose

These sample logs are included to:

- Demonstrate the structure of Cowrie JSON logs.
- Help users understand the input used by the AI monitoring script.
- Allow testing without deploying a complete honeypot environment.

## Note

The logs included in this repository are sample data intended for educational and demonstration purposes only.

No sensitive or personal information has been included.
